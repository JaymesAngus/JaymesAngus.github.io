photometry pipeline

I am currently working in collaboration with Juan Santisteban of the University of St Andrews on a photometry pipeline which will produce lightcurves for the data produced by the OPTICAM system.

The overarching goal is to produce a python script that will load a series of fits images, perform aperture photometry on the detected sources, and then automate a differential photometry process with the selection of a target and reference stars.

Progress

As of July 12th, we have a working pipeline which can identify sources and perform aperture photometry on each before generating a region file to allow the user to select the target and reference stars for differential photometry.
The next stage is to automate the aperture radii with a greater accuracy using growth curves and identify highly variable sources.
